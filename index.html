Math and Definitions 
w_t = arg min_{w in W} -μ̂_t^T w + γ w^T Σ̂_t w + λ_tc TC(w, w_{t-1})
subject to c_k(w) ≤ 0,
           CVaR_α(w) ≤ τ_α,
           Drawdown(w) ≤ τ_dd

v_t = arg min_v 0.5 ||v - g_new||^2
subject to <v, g_anchor> ≥ -ε

g_new fits drift, g_anchor enforces retention, ε sets allowed regression.

Deployment Path
  1. Run PEFT-only update with frozen backbone.
  2. Evaluate drift fit, anchor regression, and risk proxies.
  3. Promote only when gate thresholds pass.
  4. Rollback on regression alarms.
